# 🛒 Amazon E-commerce Data Intelligence & Interactive Dashboard

An end-to-end data analytics project conducted during the [NTI] program. This project focuses on analyzing Amazon e-commerce data to extract meaningful consumer behavior insights, optimize sales, and present KPIs through an interactive dashboard and a deployed web application.

## 🛠️ Tech Stack & Tools

* Language: Python 🐍
* Data Libraries: Pandas, NumPy
* Data Visualization: Matplotlib, Seaborn (with viridis palette 🎨)
* Web Deployment: [vercel]
* Dashboarding Tool: [vercel]

---

## 📊 Key Features & Analysis

1. Exploratory Data Analysis (EDA): Cleaned, preprocessed, and analyzed large-scale Amazon datasets to identify sales trends and product performance.
2. Interactive Web App: A user-friendly interface displaying advanced data distribution, features correlation, and automated analysis.
3. Business Dashboard: A responsive dashboard tracking key performance indicators (KPIs) like total revenue, user ratings, and buying patterns.
